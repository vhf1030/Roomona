# Roomona: 방탈출 통합 예약 확인

### 1. 기능
 - 테마별 기본 정보(위치, 매장, 별점, 제한시간, 장르, 난이도, 활동성, 추천인원, 홈페이지 주소)
 - 실시간 예약 현황 확인
 - 테마별 지표 표준화(별점, 난이도, 예약율)
 - 리뷰 지표 텍스트마이닝(공포도, 활동성, 추천인원)
 - 유저기반 테마 비교
